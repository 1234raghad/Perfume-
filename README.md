# مشروع متجر شذى العطور الفاخرة ✨

## فكرة المشروع:
موقع ويب تعريفي وتفاعلي مميز خاص بمحل عطور يعرض المنتجات الأكثر مبيعاً بشكل منظم ومتناسق مع شاشات الهواتف، ويوفر ميزة تفاعلية للمتصفحين لتبديل مظهر الموقع، بالإضافة إلى استمارة حجز ذكية تتحقق من صحة البيانات قبل الإرسال.

## الأدوات المستخدمة في التطوير:
- **HTML5**: لبناء وتوزيع الهيكل الأساسي للصفحة والعناصر.
- **CSS3**: لتنسيق الألوان الفخمة، الخطوط، والمسافات وضمان جمالية التصميم.
- **JavaScript**: لإضافة ميزات التفاعل الديناميكي (الوضع المظلم وفحص المدخلات).
